# Exemplos de configuração

Arquivos sintéticos prontos para testar a pipeline completa via UI ou API:

| Arquivo                       | Vendor   | Modelo    |
|-------------------------------|----------|-----------|
| `fiberhome_an5516_basic.cfg`  | Fiberhome | AN5516    |
| `zte_c600_basic.cfg`          | ZTE      | C600/C300 |
| `huawei_ma5800_basic.cfg`     | Huawei   | MA5800    |
| `datacom_dm4615_basic.cfg`    | Datacom  | DM4615    |

Configurações reais (do operador) podem ser encontradas em `modelos-config/`
no diretório raiz do projeto, fora deste pacote — elas servem de referência
para os parsers.
